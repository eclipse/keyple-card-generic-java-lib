This document is the specification of the generic card extension aiming to use the Java Keyple service and Keyple plugins without targeting a specific card technology.

Overview

Use {@link org.eclipse.keyple.card.generic.GenericExtensionServiceProvider#getService()} to get the unique instance of the {@link org.eclipse.keyple.card.generic.GenericExtensionService} interface containing all the entry points provided by the extension:

  • Creation of a selection case targeting a specific card to be provided to the Keyple Selection Service.
    {@link org.eclipse.keyple.card.generic.GenericExtensionService#createCardSelection(CardSelector cardSelector)}
    A selection case targets a specific card defined by an instance of {@link org.calypsonet.terminal.reader.selection.spi.CardSelector}.
  • Transaction service to perform APDU exchanges with a card.
    {@link org.eclipse.keyple.card.generic.GenericExtensionService#createCardTransaction(CardReader reader, SmartCard card)}
